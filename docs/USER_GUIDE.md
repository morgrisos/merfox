# MerFox ユーザーガイド

> 対象バージョン: v0.1.96  
> 対象OS: macOS / Windows 10以上

---

## 0. 最初にやること

1. ライセンスキーを受け取る（管理者から別途送付）
2. MerFox をインストールする（→ 1章）
3. ライセンスを認証する（→ 2章）
4. メルカリの検索URLをコピーしておく

---

## 1. インストール

### Mac

1. `MerFox-0.1.96-mac-arm64.dmg` をダブルクリック
2. MerFox.app を Applications フォルダにドラッグ
3. 初回起動時に「開発元を確認できません」と出たら:
   - システム設定 → プライバシーとセキュリティ → 「このまま開く」をクリック

### Windows

1. `MerFox-0.1.96-win-x64.exe` をダブルクリック
2. インストーラーの案内に従って「次へ」を押す
3. 「WindowsによってPCが保護されました」と出たら:
   - 「詳細情報」→「実行」をクリック
4. ファイアウォールの確認が出たら「許可」を選択

---

## 2. ライセンス認証

1. 左サイドバーの「設定」をクリック
2. ライセンスキー欄に受け取ったキー（例: `MERFOX-XXXX-XXXX-XXXX`）を貼り付け
3. 「認証」ボタンを押す
4. 「ライセンス認証済み」と表示されれば完了

> **認証に失敗する場合**  
> - キーを正確にコピーしているか確認（前後にスペースがないか）  
> - インターネット接続を確認  
> - 解決しない場合は管理者に連絡

## 3. 基本の使い方（メルカリ→Amazon TSV生成）

### 全体の流れ

```
ステップ1: メルカリ検索URLを入力
ステップ2: 対象を絞り込む（キーワード・価格フィルター）
ステップ3: 抽出実行（自動でメルカリを巡回）
ステップ4: 結果を確認
ステップ5: Amazon ASIN を設定する（カテゴリ変換）
ステップ6: Amazon用TSVをダウンロード
```

### ステップ詳細

**ステップ1〜3（ウィザード）**
1. ダッシュボードの「新規リサーチを開始」をクリック
2. メルカリの検索結果ページのURL を貼り付ける
   - 例: `https://jp.mercari.com/search?keyword=PS5&category_id=...`
3. フィルター条件（最低価格・NGワードなど）を設定して「次へ」
4. 「抽出開始」を押すと自動でメルカリを巡回（数分かかります）

**ステップ5（カテゴリ変換 / ASIN設定）**
1. 左サイドバーの「カテゴリ変換」をクリック
2. 各商品の「ASIN」欄にAmazonの商品ASINを入力

   **Amazon URLをそのまま貼り付けてもOK**  
   `https://www.amazon.co.jp/...../dp/B0BPLF9G62/ref=...`  
   → 自動でASIN（B0BPLF9G62）を抽出します

3. 入力し終えたら「保存」ボタンを押す
4. 画面右上に「保存済み ✓」と表示されれば完了

**ステップ6（TSVダウンロード）**
1. 左サイドバーの「ウィザード」→ ステップ6 に進む
2. 「最終TSVをダウンロード」ボタンを押す
3. ダウンロードした `.tsv` ファイルをAmazonセラーセントラルにアップロード

## 4. 在庫監視

メルカリの商品が売り切れたり価格が変わったら通知されます。

### 商品を登録する

1. 左サイドバーの「在庫監視」をクリック
2. 「手動追加」ボタンを押す
3. 以下を入力:
   - Amazon SKU（出品管理番号）
   - Amazon ASIN（Amazonの商品ID）
   - Amazon 商品名
   - メルカリの商品URL
4. 「追加」を押す

**CSV一括登録も可能**  
フォーマット: `SKU,ASIN,メルカリURL,商品名`

### 状態をチェックする

「状態をチェック」ボタンを押すと、登録した全商品のメルカリ状態を確認します。

| 表示 | 意味 |
|---|---|
| 初回待ち | まだ一度もチェックしていない |
| 正常 (Active) | 出品中 |
| 売り切れ | 購入が必要 |
| 削除済み | 出品が消えた |
| 確認不可 | アクセスできなかった |

---

## 5. 定期実行

メルカリの検索URLを登録しておくと、自動で定期的に新着商品を抽出します。

### ジョブを作成する

1. 左サイドバーの「定期実行」をクリック
2. 「監視ジョブを作成」ボタンを押す
3. メルカリの検索URL と 巡回間隔（推奨: 30分）を設定
4. 「保存して開始」を押す

### ジョブの操作

| ボタン | 動作 |
|---|---|
| ▶ / ■ | 開始 / 停止 |
| ✎ | URL・間隔を編集 |
| 🗑 | 削除 |

## 6. よくある質問

**Q: 「ライセンスが必要です」と出て操作できない**  
A: 設定ページでライセンス認証を完了してください（→ 2章）

**Q: 抽出が終わらない / 止まっている**  
A: メルカリの混雑時は時間がかかります。5分以上動かない場合はリサーチを停止して再試行してください

**Q: ASIN を入力したのに保存されない**  
A: 入力後に必ず「保存」ボタンを押してください。画面右上の「保存済み ✓」を確認してください

**Q: TSVをダウンロードしたが中身が空**  
A: ASIN が1件も設定されていない可能性があります。カテゴリ変換画面で「Pending: 0」になっているか確認してください

**Q: 別のパソコンで使いたい**  
A: ライセンスはデバイスに紐づいています。管理者にデバイスリセットを依頼してください

## 7. トラブル時の確認先

### ログファイルの場所

| OS | パス |
|---|---|
| Mac | `~/Library/Application Support/merfox/MerFox/merfox-boot.log` |
| Windows | `C:\Users\{ユーザー名}\AppData\Roaming\MerFox\merfox-boot.log` |

### 成果物ファイルの場所

| OS | パス |
|---|---|
| Mac | `~/Library/Application Support/merfox/MerFox/runs/{実行ID}/` |
| Windows | `C:\Users\{ユーザー名}\AppData\Roaming\MerFox\MerFox\runs\{実行ID}\` |

### アプリ内から確認

「リサーチ履歴」→「詳細レポート」→「フォルダを開く」で成果物フォルダを直接開けます

### 手動アップデートのやり方

1. 設定ページ → 「アップデートを確認」ボタン
2. 新バージョンがあれば GitHub Releases ページが開く
3. 最新の `.dmg` または `.exe` をダウンロードして再インストール
4. 保存データ（ライセンス・ジョブ・在庫監視）はアップデート後も保持されます
